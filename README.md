# 🚀 Code Shi: Performance Portfolio

A high-performance repository containing 3 major projects built with Speed, Security, and Modern Engineering in mind.

## 🛡️ Projects Overview

### 1. **Sentinel AI**
- **Goal:** Autonomous Security Remediation Agent for GitLab.
- **Brain:** Claude 3.5 Sonnet.
- **Action:** Intercepts Merge Requests, patches vulnerabilities, and creates shadow branches automatically.
- **Tech:** Node.js, GitLab API, Anthropic SDK.

### 2. **WASM-Zip Alpha**
- **Goal:** High-speed streaming file compressor for the browser.
- **Engine:** fflate (C++/WASM bit-level engine).
- **Pro Features:** AES-256 Military encryption, Multi-thread acceleration.
- **Edge:** Handles 4GB+ files without crashing the tab.

### 3. **MMA Gym Website (Maidaan MMA)**
- **Goal:** A premium, minimalist, reactive brand website for a Martial Arts Academy.
- **UI:** Custom React + CSS animations.
- **Status:** Fully responsive.

## 🛠️ Global Stack
- Full Stack: React, Node.js, Express.
- AI Logic: Anthropic Claude 3.5 / WebCrypto.
- Infrastructure: GitLab CI/CD, Vite.
